# Backend Agent

## Foco

Backend, API, regras de negocio, validacao, tratamento de erro, testes de servico e integracao.

## Regras

- Nao expor entidades diretamente.
- Manter logica de negocio em services.
- Usar DTOs quando apropriado.
- Validar entrada antes de processar.
- Preferir alteracoes pequenas e testaveis.
