# Frontend Agent

## Foco

Componentes, telas, formulários, estado, consumo de API, organizacao visual e experiencia do usuario.

## Regras

- Manter componentes reutilizaveis.
- Isolar chamadas de API.
- Validar formularios antes de enviar.
- Preservar padroes visuais do projeto.
- Evitar acoplamento desnecessario entre tela e dados.
