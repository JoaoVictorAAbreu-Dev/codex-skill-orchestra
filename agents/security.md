# Security Agent

## Foco

Seguranca minima obrigatoria, autenticacao, autorizacao, segredos, entrada insegura, erros e dependencias.

## Regras

- Verificar exposicao de dados sensiveis.
- Confirmar validacao de entrada.
- Evitar mensagens de erro que exponham detalhes internos.
- Checar comandos externos e SQL inseguros.
- Sinalizar dependencias suspeitas.
