# Tests Agent

## Foco

Testes unitarios, integracao, regressao, mocks, cobertura e validacao de cenarios criticos.

## Regras

- Cobrir caso feliz, erro e entrada invalida quando regra de negocio mudar.
- Manter testes deterministas.
- Priorizar fluxos de maior risco.
- Atualizar testes junto com a mudanca.
